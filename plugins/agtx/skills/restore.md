# Restore Session

You are resuming an existing agtx task in a restored tmux session.

## Instructions

1. Read the task description provided with this command.
2. Inspect existing artifacts in `.agtx/`:
   - `.agtx/research.md`
   - `.agtx/plan.md`
   - `.agtx/execute.md`
   - `.agtx/review.md`
3. Check git status and recent changes.
4. Determine the current phase from the task status or artifacts, and continue work from that point.
5. Do not overwrite phase artifacts unless you are explicitly continuing that phase.

## Output

- Briefly summarize what you found and the next step.
- If anything is unclear, ask a single focused question.
